# Licenses
(*Licenses*)

## Overview

Actions related to licenses. The <code>/licenses</code> endpoints do not apply to Cribl.Cloud deployments.

### Available Operations

* [ListLicense](#listlicense) - Get a list of License objects
* [CreateLicense](#createlicense) - Add a license to your deployment
* [GetLicenseByID](#getlicensebyid) - Get License by ID
* [DeleteLicenseByID](#deletelicensebyid) - Delete License
* [GetLicense](#getlicense) - Get license usage metrics, aggregated by day, up to last 90 days
