# Auth
(*Auth*)

## Overview

Actions related to authentication. Do not use the /auth endpoints in Cribl.Cloud deployments. Instead, follow the instructions at https://docs.cribl.io/stream/api-tutorials/#criblcloud to authenticate for Cribl.Cloud.

### Available Operations

* [GetAuthAuthorizationCodeCallback](#getauthauthorizationcodecallback) - API call that the IDP should use for an authorization code callback
* [GetAuthMetadata](#getauthmetadata) - Obtain metadata which Cribl Stream/Edge uses when acting as a Service Provider
* [Login](#login) - Log in and obtain Auth token
* [Logout](#logout) - Log current user out
* [CreateAuthSloCallback](#createauthslocallback) - API call that the IDP should use for a logout request
* [GetAuthSloCallback](#getauthslocallback) - Accepts a logout request from an IDP and logs out the user
* [GetAuthSlo](#getauthslo) - Redirect user to IDP with logout request
* [CreateAuthSsoCallback](#createauthssocallback) - API call that the IDP should use for an authentication request
* [GetAuthSsoCallback](#getauthssocallback) - Accepts an authentication request from an IDP and authenticates the user
* [GetAuthSso](#getauthsso) - Obtain redirect information
* [GetAuthGroups](#getauthgroups) - List the external authentication system's groups
* [GetAuthMultiFactor](#getauthmultifactor) - Get PIV configuration
* [DeleteAuthUsersTokenByID](#deleteauthuserstokenbyid) - Invalidate token(s) for user *id*
* [DeleteAuthUsersInvalidationByID](#deleteauthusersinvalidationbyid) - Remove invalidation for user ID
