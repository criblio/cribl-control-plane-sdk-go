# CollectorRestAuthentication2

Authentication method for Discover and Collect REST calls. You can specify API key–based authentication by adding the appropriate Collect headers.


## Values

| Name                                            | Value                                           |
| ----------------------------------------------- | ----------------------------------------------- |
| `CollectorRestAuthentication2None`              | none                                            |
| `CollectorRestAuthentication2Basic`             | basic                                           |
| `CollectorRestAuthentication2BasicSecret`       | basicSecret                                     |
| `CollectorRestAuthentication2Login`             | login                                           |
| `CollectorRestAuthentication2LoginSecret`       | loginSecret                                     |
| `CollectorRestAuthentication2Oauth`             | oauth                                           |
| `CollectorRestAuthentication2OauthSecret`       | oauthSecret                                     |
| `CollectorRestAuthentication2GoogleOauth`       | google_oauth                                    |
| `CollectorRestAuthentication2GoogleOauthSecret` | google_oauthSecret                              |
| `CollectorRestAuthentication2Hmac`              | hmac                                            |
