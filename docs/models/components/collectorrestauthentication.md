# CollectorRestAuthentication

Authentication method for Discover and Collect REST calls. You can specify API key–based authentication by adding the appropriate Collect headers.


## Values

| Name                                           | Value                                          |
| ---------------------------------------------- | ---------------------------------------------- |
| `CollectorRestAuthenticationNone`              | none                                           |
| `CollectorRestAuthenticationBasic`             | basic                                          |
| `CollectorRestAuthenticationBasicSecret`       | basicSecret                                    |
| `CollectorRestAuthenticationLogin`             | login                                          |
| `CollectorRestAuthenticationLoginSecret`       | loginSecret                                    |
| `CollectorRestAuthenticationOauth`             | oauth                                          |
| `CollectorRestAuthenticationOauthSecret`       | oauthSecret                                    |
| `CollectorRestAuthenticationGoogleOauth`       | google_oauth                                   |
| `CollectorRestAuthenticationGoogleOauthSecret` | google_oauthSecret                             |
| `CollectorRestAuthenticationHmac`              | hmac                                           |
