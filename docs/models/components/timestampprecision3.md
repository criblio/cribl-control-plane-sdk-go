# TimestampPrecision3

Sets the precision for the supplied Unix time values. Defaults to milliseconds.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `TimestampPrecision3Ns` | ns                      |
| `TimestampPrecision3U`  | u                       |
| `TimestampPrecision3Ms` | ms                      |
| `TimestampPrecision3S`  | s                       |
| `TimestampPrecision3M`  | m                       |
| `TimestampPrecision3H`  | h                       |
