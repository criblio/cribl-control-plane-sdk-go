# TimestampPrecision9

Sets the precision for the supplied Unix time values. Defaults to milliseconds.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `TimestampPrecision9Ns` | ns                      |
| `TimestampPrecision9U`  | u                       |
| `TimestampPrecision9Ms` | ms                      |
| `TimestampPrecision9S`  | s                       |
| `TimestampPrecision9M`  | m                       |
| `TimestampPrecision9H`  | h                       |
