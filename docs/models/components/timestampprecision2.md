# TimestampPrecision2

Sets the precision for the supplied Unix time values. Defaults to milliseconds.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `TimestampPrecision2Ns` | ns                      |
| `TimestampPrecision2U`  | u                       |
| `TimestampPrecision2Ms` | ms                      |
| `TimestampPrecision2S`  | s                       |
| `TimestampPrecision2M`  | m                       |
| `TimestampPrecision2H`  | h                       |
