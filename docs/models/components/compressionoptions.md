# CompressionOptions

Codec to use to compress the data before sending to Kafka


## Values

| Name                       | Value                      |
| -------------------------- | -------------------------- |
| `CompressionOptionsNone`   | none                       |
| `CompressionOptionsGzip`   | gzip                       |
| `CompressionOptionsSnappy` | snappy                     |
| `CompressionOptionsLz4`    | lz4                        |
| `CompressionOptionsZstd`   | zstd                       |
