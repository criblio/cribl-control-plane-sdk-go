# CollectorHealthCheckAuthentication3

Authentication method for Discover and Collect REST calls. You can specify API Key–based authentication by adding the appropriate Collect headers.


## Values

| Name                                             | Value                                            |
| ------------------------------------------------ | ------------------------------------------------ |
| `CollectorHealthCheckAuthentication3None`        | none                                             |
| `CollectorHealthCheckAuthentication3Basic`       | basic                                            |
| `CollectorHealthCheckAuthentication3BasicSecret` | basicSecret                                      |
| `CollectorHealthCheckAuthentication3Login`       | login                                            |
| `CollectorHealthCheckAuthentication3LoginSecret` | loginSecret                                      |
| `CollectorHealthCheckAuthentication3Oauth`       | oauth                                            |
| `CollectorHealthCheckAuthentication3OauthSecret` | oauthSecret                                      |
