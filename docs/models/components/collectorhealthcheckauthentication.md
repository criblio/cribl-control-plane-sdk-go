# CollectorHealthCheckAuthentication

Authentication method for Discover and Collect REST calls. You can specify API Key–based authentication by adding the appropriate Collect headers.


## Values

| Name                                            | Value                                           |
| ----------------------------------------------- | ----------------------------------------------- |
| `CollectorHealthCheckAuthenticationNone`        | none                                            |
| `CollectorHealthCheckAuthenticationBasic`       | basic                                           |
| `CollectorHealthCheckAuthenticationBasicSecret` | basicSecret                                     |
| `CollectorHealthCheckAuthenticationLogin`       | login                                           |
| `CollectorHealthCheckAuthenticationLoginSecret` | loginSecret                                     |
| `CollectorHealthCheckAuthenticationOauth`       | oauth                                           |
| `CollectorHealthCheckAuthenticationOauthSecret` | oauthSecret                                     |
