# TimestampPrecision5

Sets the precision for the supplied Unix time values. Defaults to milliseconds.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `TimestampPrecision5Ns` | ns                      |
| `TimestampPrecision5U`  | u                       |
| `TimestampPrecision5Ms` | ms                      |
| `TimestampPrecision5S`  | s                       |
| `TimestampPrecision5M`  | m                       |
| `TimestampPrecision5H`  | h                       |
