# DiscoverTypeRest

Defines how task discovery will be performed. Each entry returned by the Discover operation will result in a Collect task.


## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `DiscoverTypeRestHTTP` | http                   |
| `DiscoverTypeRestJSON` | json                   |
| `DiscoverTypeRestList` | list                   |
| `DiscoverTypeRestNone` | none                   |
