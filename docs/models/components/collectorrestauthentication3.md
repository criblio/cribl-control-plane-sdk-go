# CollectorRestAuthentication3

Authentication method for Discover and Collect REST calls. You can specify API key–based authentication by adding the appropriate Collect headers.


## Values

| Name                                            | Value                                           |
| ----------------------------------------------- | ----------------------------------------------- |
| `CollectorRestAuthentication3None`              | none                                            |
| `CollectorRestAuthentication3Basic`             | basic                                           |
| `CollectorRestAuthentication3BasicSecret`       | basicSecret                                     |
| `CollectorRestAuthentication3Login`             | login                                           |
| `CollectorRestAuthentication3LoginSecret`       | loginSecret                                     |
| `CollectorRestAuthentication3Oauth`             | oauth                                           |
| `CollectorRestAuthentication3OauthSecret`       | oauthSecret                                     |
| `CollectorRestAuthentication3GoogleOauth`       | google_oauth                                    |
| `CollectorRestAuthentication3GoogleOauthSecret` | google_oauthSecret                              |
| `CollectorRestAuthentication3Hmac`              | hmac                                            |
