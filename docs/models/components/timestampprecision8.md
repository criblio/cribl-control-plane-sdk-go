# TimestampPrecision8

Sets the precision for the supplied Unix time values. Defaults to milliseconds.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `TimestampPrecision8Ns` | ns                      |
| `TimestampPrecision8U`  | u                       |
| `TimestampPrecision8Ms` | ms                      |
| `TimestampPrecision8S`  | s                       |
| `TimestampPrecision8M`  | m                       |
| `TimestampPrecision8H`  | h                       |
