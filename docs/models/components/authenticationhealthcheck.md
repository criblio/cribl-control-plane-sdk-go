# AuthenticationHealthCheck

Authentication method for Discover and Collect REST calls. You can specify API Key–based authentication by adding the appropriate Collect headers.


## Values

| Name                                   | Value                                  |
| -------------------------------------- | -------------------------------------- |
| `AuthenticationHealthCheckNone`        | none                                   |
| `AuthenticationHealthCheckBasic`       | basic                                  |
| `AuthenticationHealthCheckBasicSecret` | basicSecret                            |
| `AuthenticationHealthCheckLogin`       | login                                  |
| `AuthenticationHealthCheckLoginSecret` | loginSecret                            |
| `AuthenticationHealthCheckOauth`       | oauth                                  |
| `AuthenticationHealthCheckOauthSecret` | oauthSecret                            |
