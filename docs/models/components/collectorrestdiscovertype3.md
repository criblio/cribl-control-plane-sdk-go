# CollectorRestDiscoverType3

Defines how task discovery will be performed. Each entry returned by the Discover operation will result in a Collect task.


## Values

| Name                             | Value                            |
| -------------------------------- | -------------------------------- |
| `CollectorRestDiscoverType3HTTP` | http                             |
| `CollectorRestDiscoverType3JSON` | json                             |
| `CollectorRestDiscoverType3List` | list                             |
| `CollectorRestDiscoverType3None` | none                             |
