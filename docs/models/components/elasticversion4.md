# ElasticVersion4

Optional Elasticsearch version, used to format events. If not specified, will auto-discover version.


## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `ElasticVersion4Auto`  | auto                   |
| `ElasticVersion4Six`   | 6                      |
| `ElasticVersion4Seven` | 7                      |
