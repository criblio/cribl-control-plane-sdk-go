# TimestampPrecision7

Sets the precision for the supplied Unix time values. Defaults to milliseconds.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `TimestampPrecision7Ns` | ns                      |
| `TimestampPrecision7U`  | u                       |
| `TimestampPrecision7Ms` | ms                      |
| `TimestampPrecision7S`  | s                       |
| `TimestampPrecision7M`  | m                       |
| `TimestampPrecision7H`  | h                       |
