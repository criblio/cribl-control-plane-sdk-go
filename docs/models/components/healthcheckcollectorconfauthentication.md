# HealthCheckCollectorConfAuthentication

Authentication method for Discover and Collect REST calls. You can specify API Key–based authentication by adding the appropriate Collect headers.


## Values

| Name                                                | Value                                               |
| --------------------------------------------------- | --------------------------------------------------- |
| `HealthCheckCollectorConfAuthenticationNone`        | none                                                |
| `HealthCheckCollectorConfAuthenticationBasic`       | basic                                               |
| `HealthCheckCollectorConfAuthenticationBasicSecret` | basicSecret                                         |
| `HealthCheckCollectorConfAuthenticationLogin`       | login                                               |
| `HealthCheckCollectorConfAuthenticationLoginSecret` | loginSecret                                         |
| `HealthCheckCollectorConfAuthenticationOauth`       | oauth                                               |
| `HealthCheckCollectorConfAuthenticationOauthSecret` | oauthSecret                                         |
