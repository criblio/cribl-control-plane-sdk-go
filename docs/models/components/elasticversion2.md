# ElasticVersion2

Optional Elasticsearch version, used to format events. If not specified, will auto-discover version.


## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `ElasticVersion2Auto`  | auto                   |
| `ElasticVersion2Six`   | 6                      |
| `ElasticVersion2Seven` | 7                      |
