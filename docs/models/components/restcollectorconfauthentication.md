# RestCollectorConfAuthentication

Authentication method for Discover and Collect REST calls. You can specify API key–based authentication by adding the appropriate Collect headers.


## Values

| Name                                               | Value                                              |
| -------------------------------------------------- | -------------------------------------------------- |
| `RestCollectorConfAuthenticationNone`              | none                                               |
| `RestCollectorConfAuthenticationBasic`             | basic                                              |
| `RestCollectorConfAuthenticationBasicSecret`       | basicSecret                                        |
| `RestCollectorConfAuthenticationLogin`             | login                                              |
| `RestCollectorConfAuthenticationLoginSecret`       | loginSecret                                        |
| `RestCollectorConfAuthenticationOauth`             | oauth                                              |
| `RestCollectorConfAuthenticationOauthSecret`       | oauthSecret                                        |
| `RestCollectorConfAuthenticationGoogleOauth`       | google_oauth                                       |
| `RestCollectorConfAuthenticationGoogleOauthSecret` | google_oauthSecret                                 |
| `RestCollectorConfAuthenticationHmac`              | hmac                                               |
