# AcknowledgmentsOptions

Control the number of required acknowledgments


## Values

| Name                           | Value                          |
| ------------------------------ | ------------------------------ |
| `AcknowledgmentsOptionsOne`    | 1                              |
| `AcknowledgmentsOptionsZero`   | 0                              |
| `AcknowledgmentsOptionsMinus1` | -1                             |
