# TimestampPrecision1

Sets the precision for the supplied Unix time values. Defaults to milliseconds.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `TimestampPrecision1Ns` | ns                      |
| `TimestampPrecision1U`  | u                       |
| `TimestampPrecision1Ms` | ms                      |
| `TimestampPrecision1S`  | s                       |
| `TimestampPrecision1M`  | m                       |
| `TimestampPrecision1H`  | h                       |
