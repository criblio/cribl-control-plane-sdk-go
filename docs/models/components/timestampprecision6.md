# TimestampPrecision6

Sets the precision for the supplied Unix time values. Defaults to milliseconds.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `TimestampPrecision6Ns` | ns                      |
| `TimestampPrecision6U`  | u                       |
| `TimestampPrecision6Ms` | ms                      |
| `TimestampPrecision6S`  | s                       |
| `TimestampPrecision6M`  | m                       |
| `TimestampPrecision6H`  | h                       |
