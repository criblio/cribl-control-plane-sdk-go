# TimestampPrecision10

Sets the precision for the supplied Unix time values. Defaults to milliseconds.


## Values

| Name                     | Value                    |
| ------------------------ | ------------------------ |
| `TimestampPrecision10Ns` | ns                       |
| `TimestampPrecision10U`  | u                        |
| `TimestampPrecision10Ms` | ms                       |
| `TimestampPrecision10S`  | s                        |
| `TimestampPrecision10M`  | m                        |
| `TimestampPrecision10H`  | h                        |
