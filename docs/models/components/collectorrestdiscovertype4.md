# CollectorRestDiscoverType4

Defines how task discovery will be performed. Each entry returned by the Discover operation will result in a Collect task.


## Values

| Name                             | Value                            |
| -------------------------------- | -------------------------------- |
| `CollectorRestDiscoverType4HTTP` | http                             |
| `CollectorRestDiscoverType4JSON` | json                             |
| `CollectorRestDiscoverType4List` | list                             |
| `CollectorRestDiscoverType4None` | none                             |
