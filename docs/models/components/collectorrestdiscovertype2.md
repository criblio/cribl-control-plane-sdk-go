# CollectorRestDiscoverType2

Defines how task discovery will be performed. Each entry returned by the Discover operation will result in a Collect task.


## Values

| Name                             | Value                            |
| -------------------------------- | -------------------------------- |
| `CollectorRestDiscoverType2HTTP` | http                             |
| `CollectorRestDiscoverType2JSON` | json                             |
| `CollectorRestDiscoverType2List` | list                             |
| `CollectorRestDiscoverType2None` | none                             |
