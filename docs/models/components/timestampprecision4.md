# TimestampPrecision4

Sets the precision for the supplied Unix time values. Defaults to milliseconds.


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `TimestampPrecision4Ns` | ns                      |
| `TimestampPrecision4U`  | u                       |
| `TimestampPrecision4Ms` | ms                      |
| `TimestampPrecision4S`  | s                       |
| `TimestampPrecision4M`  | m                       |
| `TimestampPrecision4H`  | h                       |
