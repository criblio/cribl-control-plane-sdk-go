# AuthenticationRest

Authentication method for Discover and Collect REST calls. You can specify API key–based authentication by adding the appropriate Collect headers.


## Values

| Name                                  | Value                                 |
| ------------------------------------- | ------------------------------------- |
| `AuthenticationRestNone`              | none                                  |
| `AuthenticationRestBasic`             | basic                                 |
| `AuthenticationRestBasicSecret`       | basicSecret                           |
| `AuthenticationRestLogin`             | login                                 |
| `AuthenticationRestLoginSecret`       | loginSecret                           |
| `AuthenticationRestOauth`             | oauth                                 |
| `AuthenticationRestOauthSecret`       | oauthSecret                           |
| `AuthenticationRestGoogleOauth`       | google_oauth                          |
| `AuthenticationRestGoogleOauthSecret` | google_oauthSecret                    |
| `AuthenticationRestHmac`              | hmac                                  |
