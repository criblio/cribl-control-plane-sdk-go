# ElasticVersion3

Optional Elasticsearch version, used to format events. If not specified, will auto-discover version.


## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `ElasticVersion3Auto`  | auto                   |
| `ElasticVersion3Six`   | 6                      |
| `ElasticVersion3Seven` | 7                      |
