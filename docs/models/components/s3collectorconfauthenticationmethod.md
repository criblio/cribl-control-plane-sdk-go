# S3CollectorConfAuthenticationMethod

AWS authentication method. Choose Auto to use IAM roles.


## Values

| Name                                        | Value                                       |
| ------------------------------------------- | ------------------------------------------- |
| `S3CollectorConfAuthenticationMethodAuto`   | auto                                        |
| `S3CollectorConfAuthenticationMethodManual` | manual                                      |
| `S3CollectorConfAuthenticationMethodSecret` | secret                                      |
