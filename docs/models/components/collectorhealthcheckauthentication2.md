# CollectorHealthCheckAuthentication2

Authentication method for Discover and Collect REST calls. You can specify API Key–based authentication by adding the appropriate Collect headers.


## Values

| Name                                             | Value                                            |
| ------------------------------------------------ | ------------------------------------------------ |
| `CollectorHealthCheckAuthentication2None`        | none                                             |
| `CollectorHealthCheckAuthentication2Basic`       | basic                                            |
| `CollectorHealthCheckAuthentication2BasicSecret` | basicSecret                                      |
| `CollectorHealthCheckAuthentication2Login`       | login                                            |
| `CollectorHealthCheckAuthentication2LoginSecret` | loginSecret                                      |
| `CollectorHealthCheckAuthentication2Oauth`       | oauth                                            |
| `CollectorHealthCheckAuthentication2OauthSecret` | oauthSecret                                      |
