# CollectorRestDiscoverType5

Defines how task discovery will be performed. Each entry returned by the Discover operation will result in a Collect task.

| Name                             | Value                            |
| -------------------------------- | -------------------------------- |
| `CollectorRestDiscoverType5HTTP` | http                             |
| `CollectorRestDiscoverType5JSON` | json                             |
| `CollectorRestDiscoverType5List` | list                             |
| `CollectorRestDiscoverType5None` | none                             |
