## Values

| Name                                       | Value                                      |
| ------------------------------------------ | ------------------------------------------ |
| `HTTPCompressCompressionOpenTelemetryNone` | none                                       |
| `HTTPCompressCompressionOpenTelemetryGzip` | gzip                                       |
