# SeveritySyslog

Default value for message severity. Will be overwritten by value of __severity if set. Defaults to notice.


## Values

| Name                  | Value                 |
| --------------------- | --------------------- |
| `SeveritySyslogZero`  | 0                     |
| `SeveritySyslogOne`   | 1                     |
| `SeveritySyslogTwo`   | 2                     |
| `SeveritySyslogThree` | 3                     |
| `SeveritySyslogFour`  | 4                     |
| `SeveritySyslogFive`  | 5                     |
| `SeveritySyslogSix`   | 6                     |
| `SeveritySyslogSeven` | 7                     |
