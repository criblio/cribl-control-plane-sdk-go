# FormatDynatraceHTTP

How to format events before sending. Defaults to JSON. Plaintext is not currently supported.


## Values

| Name                           | Value                          |
| ------------------------------ | ------------------------------ |
| `FormatDynatraceHTTPJSONArray` | json_array                     |
| `FormatDynatraceHTTPPlaintext` | plaintext                      |
