# AcknowledgmentsKafka

Control the number of required acknowledgments.


## Values

| Name                         | Value                        |
| ---------------------------- | ---------------------------- |
| `AcknowledgmentsKafkaOne`    | 1                            |
| `AcknowledgmentsKafkaZero`   | 0                            |
| `AcknowledgmentsKafkaMinus1` | -1                           |
