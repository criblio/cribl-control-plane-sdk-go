# ReportLevel

Level of ingestion status reporting. Defaults to FailuresOnly.


## Values

| Name                              | Value                             |
| --------------------------------- | --------------------------------- |
| `ReportLevelFailuresOnly`         | failuresOnly                      |
| `ReportLevelDoNotReport`          | doNotReport                       |
| `ReportLevelFailuresAndSuccesses` | failuresAndSuccesses              |
