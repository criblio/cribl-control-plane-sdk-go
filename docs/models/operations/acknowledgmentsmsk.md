# AcknowledgmentsMsk

Control the number of required acknowledgments.


## Values

| Name                       | Value                      |
| -------------------------- | -------------------------- |
| `AcknowledgmentsMskOne`    | 1                          |
| `AcknowledgmentsMskZero`   | 0                          |
| `AcknowledgmentsMskMinus1` | -1                         |
