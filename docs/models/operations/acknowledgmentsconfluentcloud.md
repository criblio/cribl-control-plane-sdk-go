# AcknowledgmentsConfluentCloud

Control the number of required acknowledgments.


## Values

| Name                                  | Value                                 |
| ------------------------------------- | ------------------------------------- |
| `AcknowledgmentsConfluentCloudOne`    | 1                                     |
| `AcknowledgmentsConfluentCloudZero`   | 0                                     |
| `AcknowledgmentsConfluentCloudMinus1` | -1                                    |
